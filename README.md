# Flowdesk AI — Workplace Productivity Assistant

A modern, responsive SaaS-style web application that helps professionals automate everyday workplace tasks using AI. Built with TanStack Start, React, TypeScript, Tailwind CSS, and the Lovable AI Gateway.

## Features

- **Smart Email Generator** — Generate professional emails from a topic, tone, audience, and key points. Supports custom sender/recipient names and optional additional context.
- **Meeting Notes Summarizer** — Paste raw meeting notes and receive a concise summary with discussion points, decisions, action items, and owners.
- **AI Task Planner** — Enter a goal, deadline, available hours, and priority to get a prioritized, time-bound action plan with milestones and task breakdowns.

Each tool includes editable AI output, copy-to-clipboard, regenerate, and clear actions, plus a responsible AI disclaimer.

## Tech Stack

- **Framework:** [TanStack Start](https://tanstack.com/start/) v1
- **Build Tool:** Vite 7
- **UI Library:** React 19, TypeScript
- **Styling:** Tailwind CSS v4
- **Components:** shadcn/ui
- **AI:** Lovable AI Gateway (`google/gemini-3-flash-preview`)
- **Icons:** Lucide React
- **Notifications:** Sonner

## Project Structure

```
src/
  components/          # Reusable UI components (sidebar, tool shell, theme toggle)
  lib/                 # Server functions and utilities
  routes/              # TanStack file-based routes
  styles.css           # Global styles and Tailwind theme tokens
  router.tsx           # Router configuration
  start.ts             # App start configuration
  server.ts            # Server entry
```

## Development

### Prerequisites

- [Bun](https://bun.sh/) (recommended)
- Node.js 20+ (if using npm/pnpm)

### Install dependencies

```bash
bun install
```

### Run the development server

```bash
bun dev
```

The app will be available at `http://localhost:8080`.

### Build for production

```bash
bun run build
```

## AI Configuration

The AI features use the Lovable AI Gateway. Ensure the `LOVABLE_API_KEY` environment variable is set in your Lovable workspace or deployment environment.

## Deployment

This project is designed for Lovable's edge hosting. It can also be deployed to any platform that supports TanStack Start / Vite, such as Vercel or Cloudflare Pages.

## Responsible AI Usage

AI-generated content may contain inaccuracies. Always review, verify, and edit all output before using it for professional communication or decision-making. Do not enter confidential or sensitive business information into AI prompts.
